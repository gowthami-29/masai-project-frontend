# 💰 FinTrack – Personal Finance Management App
## 📌 Project Title
FinTrack – Smart Personal Finance Tracker
# 📖 Project Description
FinTrack is a full-stack personal finance management application that allows users to:
Track income and expenses
Manage recurring payments
Set budgets
Split shared expenses
Monitor financial health score
Track savings goals
The application ensures user-specific data privacy and secure authentication using JWT and Supabase.
# ✨ Features
    1. 🔐 User Authentication (Login / Signup)
    2. 💵 Income Management
    3. 💳 Expense Tracking
    4. 🔁 Recurring Payments
    5. 👥 Shared Expense Splitting
    6. 📊 Financial Health Score
    7. 🎯 Savings Goals Tracker
    8. 📈 Dashboard Overview
    9. 🧾 Budget Management
    10. 🔒 User-based Data Filtering
# 🛠 Tech Stack Used
   # Frontend:
        React.js
        Vite
        Tailwind CSS
        ShadCN UI
        Axios
   # Backend:
        Node.js
        Express.js
        JWT Authentication
   # Database:
        Supabase (PostgreSQL)
   # Deployment:
        Frontend → Netlify
        Backend → Render
