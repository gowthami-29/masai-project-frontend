
💰 FinTrack – Personal Finance Management App
📌 Project Title
FinTrack – Smart Personal Finance Tracker
📖 Project Description
FinTrack is a full-stack personal finance management application that allows users to:
Track income and expenses
Manage recurring payments
Set budgets
Split shared expenses
Monitor financial health score
Track savings goals
The application ensures user-specific data privacy and secure authentication using JWT and Supabase.

🔐 User Authentication (Login / Signup)
